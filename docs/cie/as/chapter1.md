## Key Concepts

### Cell Size and Units

| Unit | Symbol | Value |
|------|--------|-------|
| Millimeter | mm | 10⁻³ m |
| Micrometer | μm | 10⁻⁶ m |
| Nanometer | nm | 10⁻⁹ m |

**Magnification Formula:**
```
Magnification = Image size / Actual size
```

### Prokaryotic vs Eukaryotic Cells

| Feature | Prokaryotic | Eukaryotic |
|---------|-------------|------------|
| Nucleus | Absent | Present |
| DNA | Circular | Linear |
| Ribosomes | 70S | 80S |
| Membrane-bound organelles | Absent | Present |
| Cell size | 0.5-5 μm | 10-100 μm |

### Animal vs Plant Cells

**Animal cells have:**
- Centrioles
- Small or no vacuoles
- No cell wall

**Plant cells have:**
- Cell wall (cellulose)
- Large central vacuole
- Chloroplasts
- No centrioles

## Study Tips

1. **Practice calculations** - magnification questions are common in exams
2. **Draw diagrams** - sketch cells to help remember structures
3. **Compare tables** - use tables to organize similarities and differences
4. **Link functions** - always connect structure to function

## Exam-Style Questions

1. Calculate the actual size of a cell if the image measures 50 mm at ×400 magnification.

2. State three differences between prokaryotic and eukaryotic cells.

3. Explain why mitochondria are more numerous in muscle cells than in skin cells.
