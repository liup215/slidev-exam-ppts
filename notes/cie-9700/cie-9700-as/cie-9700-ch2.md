## 2.1 碳水化合物 (Carbohydrates)

### 单糖 (Monosaccharides)

单糖通式：$(CH_2O)_n$，其中 $n \geq 3$

| 单糖 | 碳数 | 示例 | 功能 |
|------|------|------|------|
| 三碳糖 | 3 | 甘油醛 | 糖酵解中间产物 |
| 五碳糖 | 5 | 核糖、脱氧核糖 | 核酸成分 |
| 六碳糖 | 6 | 葡萄糖、果糖、半乳糖 | 能量来源 |

**α-葡萄糖与β-葡萄糖的区别**：

$$\alpha\text{-葡萄糖}: \text{C1 上 -OH 在环下方}$$
$$\beta\text{-葡萄糖}: \text{C1 上 -OH 在环上方}$$

### 二糖 (Disaccharides)

缩合反应（**脱水缩合**）：

$$\text{单糖}_1 + \text{单糖}_2 \rightarrow \text{二糖} + \text{H}_2\text{O}$$

| 二糖 | 组成 | 糖苷键 |
|------|------|--------|
| 麦芽糖 (Maltose) | α-葡萄糖 + α-葡萄糖 | α-1,4 |
| 蔗糖 (Sucrose) | 葡萄糖 + 果糖 | α-1,β-2 |
| 乳糖 (Lactose) | 葡萄糖 + 半乳糖 | β-1,4 |

### 多糖 (Polysaccharides)

| 多糖 | 单体 | 糖苷键 | 功能 | 存在 |
|------|------|--------|------|------|
| 淀粉 (Starch) | α-葡萄糖 | α-1,4 (直链) / α-1,6 (支链) | 储能 | 植物 |
| 糖原 (Glycogen) | α-葡萄糖 | α-1,4 (主链) / α-1,6 (分支多) | 储能 | 动物、真菌 |
| 纤维素 (Cellulose) | β-葡萄糖 | β-1,4 | 支撑 | 植物细胞壁 |
| 几丁质 (Chitin) | N-乙酰葡萄糖胺 | β-1,4 | 支撑 | 真菌细胞壁、节肢动物外骨骼 |

> **关键考点**：淀粉和糖原均由 α-葡萄糖构成，可被水解；纤维素由 β-葡萄糖构成，人体无法消化。

## 2.2 脂质 (Lipids)

### 三酰甘油 (Triglycerides)

由**甘油**（1分子）+ **脂肪酸**（3分子）通过**酯键**连接：

$$\text{甘油} + 3 \times \text{脂肪酸} \rightarrow \text{三酰甘油} + 3\text{H}_2\text{O}$$

**饱和脂肪酸 vs 不饱和脂肪酸**：

- **饱和**：无双键，链直，熔点高，室温固态（如动物脂肪）
- **不饱和**：含 C=C 双键，链弯折，熔点低，室温液态（如植物油）

### 磷脂 (Phospholipids)

磷脂是细胞膜的基本结构成分：

```
       亲水头部 (Hydrophilic head)
       ●  ← 磷酸基团 + 甘油
       |
       |  疏水尾部 (Hydrophobic tail)
      /|\
     / | \
   脂肪酸链
```

磷脂分子具有**双亲性（amphipathic）**，因此能自发形成双分子层。

### 固醇 (Steroids)

- **胆固醇**：细胞膜流动性调节；类固醇激素前体
- **性激素**（雌激素、睾酮）：信号分子
- **维生素 D**：钙吸收调节

## 2.3 蛋白质 (Proteins)

### 氨基酸结构

通用结构：

$$\text{H}_2\text{N} - \underset{\displaystyle|}{\overset{\displaystyle R}{\text{CH}}} - \text{COOH}$$

- **氨基**（$-\text{NH}_2$）：碱性基团
- **羧基**（$-\text{COOH}$）：酸性基团  
- **R 基**：决定氨基酸种类（20种）

### 肽键形成

$$\text{氨基酸}_1 + \text{氨基酸}_2 \rightarrow \text{二肽} + \text{H}_2\text{O}$$

肽键（peptide bond）：$-\text{CO}-\text{NH}-$

### 蛋白质结构层次

| 结构级别 | 描述 | 化学键 |
|---------|------|--------|
| 一级结构 | 氨基酸序列 | 肽键 |
| 二级结构 | α-螺旋、β-折叠 | 氢键 |
| 三级结构 | 多肽链的三维折叠 | 二硫键、氢键、离子键、疏水作用 |
| 四级结构 | 多个多肽链聚合 | 同三级结构 |

> **蛋白质变性（Denaturation）**：高温或极端 pH 破坏氢键等非共价键，导致空间结构改变，但肽键（一级结构）不变。

## 2.4 核酸 (Nucleic Acids)

### 核苷酸结构

核苷酸 = **磷酸基团** + **五碳糖** + **含氮碱基**

| 碱基 | 类型 | DNA | RNA |
|------|------|-----|-----|
| 腺嘌呤 A | 嘌呤 | ✓ | ✓ |
| 鸟嘌呤 G | 嘌呤 | ✓ | ✓ |
| 胞嘧啶 C | 嘧啶 | ✓ | ✓ |
| 胸腺嘧啶 T | 嘧啶 | ✓ | ✗ |
| 尿嘧啶 U | 嘧啶 | ✗ | ✓ |

### DNA 碱基配对原则

$$A = T \text{（2个氢键）}, \quad G \equiv C \text{（3个氢键）}$$

Chargaff 定律：$\frac{A}{T} = 1$，$\frac{G}{C} = 1$，即 $[A]=[T]$，$[G]=[C]$

## 2.5 检测方法 (Detection Tests)

| 物质 | 试剂 | 阳性结果 |
|------|------|----------|
| 还原糖 | 本尼迪特试剂 (Benedict's) | 砖红色沉淀 |
| 淀粉 | 碘液 (Iodine solution) | 蓝黑色 |
| 蛋白质 | 双缩脲试剂 (Biuret) | 紫色 |
| 脂肪 | 苏丹 III/IV 染液 | 橙红色/红色 |

## 考点整理

1. **缩合反应**形成单糖、氨基酸间的化学键，**水解反应**断裂化学键
2. 纤维素β-1,4 键使其具有高强度，不可被大多数生物消化
3. 蛋白质功能多样性来源于**氨基酸序列**（一级结构）决定的三维构象
4. DNA 双链反向平行（antiparallel），3'→5' 与 5'→3'
